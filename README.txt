The instance files (adapted from instances available from OR-Library - http://people.brunel.ac.uk/~mastjjb/jeb/orlib/gapinfo.html) contain the data in the following format:

number of agents (m), number of tasks (n)
 for each agent i (i = 1, ..., m):
     satisfaction (profit) for allocating task j to agent i (j = 1, ..., n)
 for each agent i (i = 1, ..., m):
     resource consumption when allocating a task j to agent i (j = 1, ..., n)
 capacity of agent i (i = 1, ..., m)
